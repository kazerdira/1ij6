API Documentation

Welcome to the TranslateAPI documentation. This guide covers everything you need to integrate our translation API into your application.

üí°

Base URL: All API requests should be made to https://api.translateapi.dev/v1

Authentication

All API requests require authentication using a Bearer token. Include your API key in the Authorization header: 

Authorization: Bearer YOUR_API_KEY
‚ö†Ô∏è

Keep your API key secret. Never expose it in client-side code or public repositories.

Quick Start

Here's a complete example to translate text from English to French:

# Using cURL
curl -X POST "https://api.translateapi.dev/v1/translate" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello, how are you?",
    "source_lang": "eng_Latn",
    "target_lang": "fra_Latn"
  }'

Response:

{
  "translated_text": "Bonjour, comment allez-vous?",
  "source_lang": "eng_Latn",
  "target_lang": "fra_Latn",
  "processing_time_ms": 187
}

Translate Text

Translate text from one language to another.

POST /translate

Request Body

Parameter Type Required Description
text string Yes Text to translate (max 5000 characters)
source_lang string Yes Source language code (e.g., eng_Latn)
target_lang string Yes Target language code (e.g., fra_Latn)

Example Request

{
  "text": "The quick brown fox jumps over the lazy dog.",
  "source_lang": "eng_Latn",
  "target_lang": "spa_Latn"
}

Translate Audio

Transcribe an audio file and translate the result to another language.

POST /transcribe

Request Body

Parameter Type Required Description
audio_base64 string Yes Base64-encoded audio data
target_lang string Yes Target language code for translation
source_lang string No Source language (auto-detected if not provided)

Supported Audio Formats

MP3, WAV, M4A, FLAC, OGG, WebM (max 25MB)

Example Response

{
  "transcription": "Hello, this is a test recording.",
  "translated_text": "Hola, esta es una grabaci√≥n de prueba.",
  "detected_language": "en",
  "target_lang": "spa_Latn",
  "processing_time_ms": 1542
}

Transcribe Only

Transcribe audio to text without translation.

POST /transcribe-only

Request Body

Parameter Type Required Description
audio_base64 string Yes Base64-encoded audio data

Example Response

{
  "transcription": "Hello, this is a test recording.",
  "detected_language": "en",
  "processing_time_ms": 892
}

Error Handling

The API returns standard HTTP status codes and JSON error responses:

Status Code Meaning
200Success
400Bad Request - Invalid parameters
401Unauthorized - Invalid or missing API key
429Rate Limited - Too many requests
500Server Error - Something went wrong

Error Response Format

{
  "error": {
    "code": "INVALID_LANGUAGE",
    "message": "Unsupported language code: xyz_Latn"
  }
}
